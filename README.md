# Java Chat App 💬

A simple real-time chat application built using Java Socket Programming and Swing GUI.

## 🚀 Features
- Real-time messaging
- Private messaging (@user)
- Image sending (Base64)
- Drag & drop images
- Online users list

## 🛠 Tech Stack
- Java (Socket Programming)
- Java Swing (GUI)

## ▶️ How to Run

### 1. Run Server
```bash
javac ChatServer.java
java ChatServer
